#技术管理心得
###生存空间
- 不管是做技术研究、平台工具还是做项目，一个技术人员和一个团队在环境里要成长，需要匹配三个概念，即“我能做的”，“我想做的”，“需要我做的”。
- 我能做 + 我想做 + 需要我 = 核心价值
- 我能做 + 我想做 + 不需要我 = 潜在价值
- 我能做 + 我不想做 + 需要我 = 例行工作
- 我能做 + 我不想做 + 不需要我 = backup
- 我不能做 + 我想做 + 需要我 = 成长空间
- 我不能做 + 我想做 + 不需要我 = 自我追求
- 我不能做 + 我不想做 + 需要我 = 无法胜任
- 我不能做 + 我不想做 + 不需要我 = 无需关注
- 好的管理者会特别注意成员的“潜在价值”和“成长空间
